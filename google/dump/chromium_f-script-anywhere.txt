<h3 xmlns="http://www.w3.org/1999/xhtml" id="sites-page-title-header" style="" align="left">
<span id="sites-page-title" dir="ltr" tabindex="-1" style="outline: none">F-Script Anywhere</span>
</h3>
<div id="sites-canvas-main" class="sites-canvas-main">
<div id="sites-canvas-main-content">
<table xmlns="http://www.w3.org/1999/xhtml" cellspacing="0" class="sites-layout-name-one-column sites-layout-hbox"><tbody><tr><td class="sites-layout-tile sites-tile-name-content-1"><div dir="ltr">F-Script Anywhere turns any Mac app into a smalltalk-y environment: You're able to click on any view to learn its type, and you can send messages to every object in the program at run time. It's very useful to get around in Chrome: If you're wondering which piece of code is responsible for the findbar, you'd just click the findbar and get the name of the class that implements it. Put that in <a href="http://cs.chromium.org">http://cs.chromium.org</a>, and you're done. This page explains how to install and use F-Script Anywhere.<div><br /></div><h3><a name="TOC-Installing-F-Script-Anywhere"></a>Installing F-Script Anywhere</h3><div><ol><li>Download <a href="http://www.fscript.org/download/F-ScriptInjectionService.zip">http://www.fscript.org/download/F-ScriptInjectionService.zip</a> and follow the instructions in the included readme.txt.</li><li>Make sure you also downloaded FScript.Framework (as described in the aforementioned readme.txt!)</li><li>If you run OSX 10.8 or later, you need to fix up the automator workflow. Edit the injection service with Automator, and make sure the "New Text File" action generates plain text. Also, ensure that the following AppleScript refers to /tmp/gdbtemp<b><i><font size="3">.txt</font></i></b> - "New Text File" now enforces an extension.</li></ol>See <a href="http://pmougin.wordpress.com/2010/01/05/the-revenge-of-f-script-anywhere/">http://pmougin.wordpress.com/2010/01/05/the-revenge-of-f-script-anywhere/</a> for more information.</div><div><br /></div><h3><a name="TOC-Using-F-Script-Anywhere"></a>Using F-Script Anywhere</h3><div>Click "Chrome-&gt;Services-&gt;Inject F-Script into application". <b>Note</b>: This won't work for programs that are running in gdb (e.g. it won't work in Chromium if you started it through Xcode).</div><div><div style="display:block;text-align:left"><br /></div><div style="display:block;text-align:left"><a href="https://www.chromium.org/developers/f-script-anywhere/Screen%20shot%202011-04-01%20at%2011.52.02%20AM.png?attredirects=0" imageanchor="1"><img border="0" src="https://www.chromium.org/_/rsrc/1301684014834/developers/f-script-anywhere/Screen%20shot%202011-04-01%20at%2011.52.02%20AM.png" /></a></div><div style="display:block;text-align:left"><br /></div>This will take about 15 seconds in Release builds and several minutes in Debug builds. </div><div><br /></div><div>Alternatively, you could do it right from gdb (the Service is an Automator script that runs gdb for you, and there are several Automator scripts floating around, some of them don't work):</div><div><div>&gt;gdb --args ./out/Release/Chromium.app/Contents/MacOS/Chromium (or wherever it is)</div><div>(gdb) r</div></div><div>.....</div><div>&lt;Ctrl-C&gt;</div><div><div>(gdb) p (char)[[NSBundle bundleWithPath:@"/Library/Frameworks/FScript.framework"] load]</div><div>(gdb) p (void)[FScriptMenuItem insertInMainMenu]</div></div><div>(gdb) c</div><div><br /></div><div><br /></div><div>When it's done, an "F-Script" menu entry will appear in the top level menu:</div><div><br /></div><div><div style="display:block;text-align:left"><a href="https://www.chromium.org/developers/f-script-anywhere/Screen%20shot%202011-04-01%20at%2011.57.03%20AM.png?attredirects=0" imageanchor="1"><img border="0" src="https://www.chromium.org/_/rsrc/1301684390932/developers/f-script-anywhere/Screen%20shot%202011-04-01%20at%2011.57.03%20AM.png" /></a></div><div style="display:block;text-align:left"><br /></div><div style="display:block;text-align:left">Click "Open Object Browser". In the window that opens, click "Select View". Say you're wondering which class implements the Omnibox, so click the Omnibox next. F-Script Anywhere will tell you that the Omnibox is an AutocompleteTextField.</div><div style="display:block;text-align:left"><br /></div><div style="display:block;text-align:left"><div style="display:block;text-align:left"><a href="https://www.chromium.org/developers/f-script-anywhere/Screen%20shot%202011-04-01%20at%204.27.30%20PM.png?attredirects=0" imageanchor="1"><img border="0" src="https://www.chromium.org/_/rsrc/1301700487127/developers/f-script-anywhere/Screen%20shot%202011-04-01%20at%204.27.30%20PM.png" /></a></div></div><div style="display:block;text-align:left"><br /></div><div style="display:block;text-align:left">You can even send methods to this object: Type "sethid" into the search box in the upper right corner, and then click "setHidden:" on the right:</div><div style="display:block;text-align:left"><br /></div><div style="display:block;text-align:left"><div style="display:block;text-align:left"><a href="https://www.chromium.org/developers/f-script-anywhere/Screen%20shot%202011-04-01%20at%2011.59.12%20AM.png?attredirects=0" imageanchor="1"><img border="0" src="https://www.chromium.org/_/rsrc/1301684546648/developers/f-script-anywhere/Screen%20shot%202011-04-01%20at%2011.59.12%20AM.png" /></a></div><div style="display:block;text-align:left"><br /></div><div style="display:block;text-align:left">Type "YES" into the popup that opens, and the Omnibox will disappear:</div><div style="display:block;text-align:left"><br /></div><div style="display:block;text-align:left"><div style="display:block;text-align:left"><a href="https://www.chromium.org/developers/f-script-anywhere/Screen%20shot%202011-04-01%20at%2011.59.21%20AM.png?attredirects=0" imageanchor="1"><img border="0" src="https://www.chromium.org/_/rsrc/1301684583957/developers/f-script-anywhere/Screen%20shot%202011-04-01%20at%2011.59.21%20AM.png" /></a></div><br /></div><br /></div>You can also navigate to a view's superview this way, and do many other fun things. Read the F-Script documentation for more. F-Script Anywhere works with every OS X app, not just with Chrome.</div></div></td></tr></tbody></table>
